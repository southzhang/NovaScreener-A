# Web Search 主力资金净流入降级方案（2026-05-26新增）

## 触发条件

当下列数据源全部不可用时，使用 web_search 获取个股前一交易日的主力净流入数据：

1. iFinD MCP `get_stock_performance` → 返回空（已排除查询格式问题）
2. push2 `stock/get` (f62字段) → VPN拦截/空响应
3. push2his `fflow/daykline/get` → VPN拦截/空响应

## 数据源

证券之星通过搜狐/网易/MSN等平台转载的每日资金流向文章，覆盖所有A股。

**搜索模式**：
```
web_search("{股票名} {代码} {日期} 主力资金净流入")
```

**搜索结果格式示例**：
```
5月25日郴电国际（600969）涨停分析：业绩扭亏、新能源落地
主力资金净流入3498.33万元，占总成交额12.95%
游资资金净流出211.41万元，占总成交额0.78%
散户资金净流出3286.93万元，占总成交额12.17%
```

**发布时间**：次日早盘前（约08:00-09:00），适合盘前/早盘分析。
**可用性**：经2026-05-26实测，对郴电国际(600969)有效。

## 字段映射

| web_search 字段 | 描述 | 单位 |
|----------------|------|------|
| 主力资金净流入 | 超大单+大单净流入合计 | 万元 |
| 占总成交额% | 主力净流入/总成交额 | % |
| 游资资金净流入 | 中单净流入 | 万元 |
| 散户资金净流入/流出 | 小单净流入 | 万元 |

## 局限性

- 只有日频数据（次日发布），无盘中实时数据
- 需要手动从搜索结果中解析数值（非结构化）
- 节假日前后数据可能延迟
- 非所有股票都有独立文章（但主力净流入较大的通常有）

## 完整案例：郴电国际(600969)

**查询**：web_search("郴电国际 600969 5月25日 主力资金净流入")
**返回**：搜狐文章链接，内容含完整资金流向数据
**结果**：主力净流入+3498.33万元(12.95%)

之前所有API(MCP+push2)均返回空。
