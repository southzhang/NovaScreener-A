# 自选池盯盘模式

当需要对一组自选股（关注池，非V10信号池）进行定期盯盘监控时使用。典型场景：用户指定3-10只关注股，每30分钟检查一次，有入场/出场时机时提醒。

## 与持仓盯盘的区别

| 维度 | 持仓盯盘 | 自选池盯盘 |
|------|----------|-----------|
| 对象 | 已持仓股票 | 关注但未持仓（或部分持仓） |
| 目的 | 风险预警+止损提醒 | 寻找入场时机 |
| 核心信号 | 跌破止损/基本面恶化 | 回踩支撑位/放量突破/催化剂出现 |
| 数据频率 | 每10-30分钟 | 每30分钟 |
| 不替代 | V10选股信号 | V10选股信号 |

## 数据采集三步法

### Step 1: 腾讯实时行情（快、可靠）

```bash
# 同时获取大盘指数+自选股行情
curl -s "https://qt.gtimg.cn/q=sh000001,sz300065,sh600584,sz300199" | iconv -f gbk -t utf-8
```

**关键字段解析**：
| 字段 | 索引 | 用途 |
|------|------|------|
| 现价 | parts[3] | 持仓盈亏计算 |
| 昨收 | parts[4] | 涨跌幅基准 |
| 成交量(手) | parts[6] | 量能判断 |
| 外盘(主动买) | parts[7] | 买卖压力 |
| 内盘(主动卖) | parts[8] | 买卖压力 |
| 涨跌幅(%) | parts[32] | 异动判断 |
| 最高 | parts[33] | 压力位参考 |
| 最低 | parts[34] | 支撑位参考 |
| 量价额组合 | parts[37] | 成交额解析（split('/')[2]/1e8=亿） |
| 换手率(%) | parts[38] | 活跃度判断 |

**成交额解析**：
```python
combo = parts[37]  # e.g. "24.50/216253/532897902"
amount_yuan = float(combo.split('/')[2])  # 532897902 元
amount_yi = amount_yuan / 100000000       # 5.33 亿
```

### Step 2: 腾讯K线获取近期支撑/压力

```bash
# 获取近10日K线（前复权）
curl -s "https://web.ifzq.gtimg.cn/appstock/app/fqkline/get?param=sz300065,day,,,10,qfq"
```

**返回格式**：JSON，`data[code]["qfqday"]` 数组，每根K线 `[日期,开盘,收盘,最高,最低,成交量]`

**支撑/压力识别方法**：
```python
# 从近10日K线找支撑和压力
recent_lows = [float(k[4]) for k in klines[-10:]]  # 最低价
recent_highs = [float(k[3]) for k in klines[-10:]]  # 最高价
support = min(recent_lows)        # 近期最低点=强支撑
resistance = max(recent_highs)    # 近期最高点=强压力
secondary_support = sorted(recent_lows)[1]  # 次低点=次支撑
```

**关键判断**：
- 多次触及同一价位不破 → 强支撑/压力
- 最低价逐渐抬高 → 上升趋势
- 最高价逐渐降低 → 下降趋势

## 入场信号判断框架

### 已持仓股票（如海兰信）

| 信号 | 条件 | 行动 |
|------|------|------|
| 止损预警 | 现价接近止损线(距止损<5%) | ⚠️ 立即提醒 |
| 回踩支撑 | 现价触及支撑位+缩量 | 👀 观察是否企稳 |
| 跌破支撑 | 现价跌破近期支撑 | ⚠️ 提高风险等级 |
| 冲高回落 | 外盘<内盘+长上影 | ⚠️ 短期压力 |

### 未持仓股票（如长电科技/翰宇药业）

| 信号 | 条件 | 入场建议 |
|------|------|----------|
| 放量突破 | 涨幅>3%+换手>5%+外盘>内盘 | 突破位回踩确认后入场 |
| 缩量回调 | 从高点回落5-10%+缩量 | 支撑位区间入场 |
| 催化剂+回踩 | 重大利好后回调至支撑 | ✅ 最佳入场点 |
| 追高警告 | 连涨>5日/涨幅>30% | ⚠️ 不追高，等回调 |

### 大盘环境过滤

```
上证>4100 → 安全，可正常推荐入场
3900-4100 → 谨慎，降低推荐力度
<3900 → 危险，不主动推荐入场
上证跌>2% → 不推荐入场
```

2. 股票B — 入场时机分析（具体价位区间）
3. 股票C — 入场时机分析（具体价位区间）

- 入场建议必须给出具体价位区间（如"23-24元区间可入场"）

## 铁律

1. **入场建议必须给具体价位区间**，不能只说"可以买了"
2. **不替代V10选股信号**，但在股价有重大异动时可提醒
3. **大盘环境不好（上证跌>2%）时不主动推荐入场**
4. **追高是最常见的错误**，连涨后必须等回调
5. **催化剂要有实质内容**（订单/临床/合作），不是纯概念炒作
