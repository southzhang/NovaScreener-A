# 同花顺10jqka资金流向页面 — 静态数据提取模式（2026-05-22确认）

## 现象

当 iFinD MCP 断连、东方财富 push2/datacenter 全部失效时，**同花顺个股资金流向页面** (`stockpage.10jqka.com.cn/<code>/Funds`) 可作为主力资金净流入的可靠降级数据源。

## 关键发现

该页面是**静态HTML**（不是JS动态渲染），web_extract 可以直接提取完整的日度资金流向数据。

## 数据内容

页面包含近5个交易日的主力资金流向表格，每行数据格式：

日期 | 主力净流入(万元) | 涨跌幅 | 主力净流入额占比(%)

其中「主力」= 大单（超大单+大单）的资金净流入。

## 提取方式

```
from hermes_tools import web_extract
result = web_extract(urls=["http://stockpage.10jqka.com.cn/301026/Funds"])
# 返回的 markdown 文本包含近5日主力资金数据表格
```

返回的 content 字段包含类似：
2026-05-18  +927.17万  +5.91%  ...
2026-05-19  +2043.69万  +4.54%  ...

## 数据版本冲突风险（2026-05-26发现）

**同花顺资金流向页面可能同时展示两个版本的数据**，收盘价和资金流数值不同：

| 版本 | 日期 | 收盘价 | 涨跌幅 | 大单净额(万) | 特征 |
|------|------|--------|--------|------------|------|
| 版本A（早盘快照） | 2026-05-26 | 23.50 | -4.16% | -5,514 | 表头含"资金净流入"和"大单净额"，无中单/小单拆分 |
| 版本B（修正数据） | 2026-05-26 | 23.08 | -5.87% | -17,908 | 表头含"总净流入"+"大单净额"+"中单净额"+"小单净额"，更完整 |

**按页面描述，第二个表格（版本B）更靠后，可能为修正数据**。但两个版本的涨跌幅都与腾讯实时收盘价23.37/-4.69%有差距。

**推荐做法**：
1. 优先使用**数据更完整**的版本（含中单/小单拆分的版本B）
2. 用腾讯行情field[50]交叉验证**方向**（正/负）
3. 不依赖绝对数值做精确决策，仅用于趋势判断
4. 如果需要精确数值，等iFinD MCP恢复后查询

## 与 iFinD MCP 数据一致性

2026-05-22 实测对比：
- 10jqka 页面：5日累计+5146.08万（每日明细可查）
- 证券之星新闻文章：5/21主力净卖出3840.22万元（⚠️ 同一日数据差异大！不同平台对「主力」定义不同）

## 局限性

| 维度 | 10jqka资金流向页面 | iFinD MCP |
|------|-------------------|-----------|
| 主力净流入（近5日） | ✅ 静态HTML，web_extract可靠 | ✅ 更详细（含区间汇总） |
| 龙虎榜 | ❌ 需JS渲染，web_extract无法抓取 | ✅ MCP查询 |
| 大宗交易 | ❌ 页面不含 | ✅ MCP查询 |
| 融资融券 | ❌ 页面不含 | ✅ MCP查询 |
| 行业板块数据 | ❌ 仅单只股票 | ✅ 全文搜索/板块分析 |
| 历史数据（>5日） | ❌ 页面仅展示近5日 | ✅ 可指定任意时间区间 |

## 适用场景

1. **iFinD MCP断连时的降级方案** — 主力净流入数据不可少（选股五关验证之一）
2. **单只持仓深度分析** — 结合 push2 f162/f164（手数+占比）交叉验证
3. **快速检查某只股票的资金面** — 无需MCP/Auth，浏览器即可查看

## 不适用场景

1. **批量查询** — 每只股票需单独请求，不适合>5只的批量场景
2. **盘中实时数据**（交易时段15:00前） — 页面的"实时资金流向"部分不会在盘中有值（显示"--"）。但"近30个交易日历史数据"表格在**收盘后（约15:30-16:00）即更新当日数据**，可用于盘后分析
3. **精确金额** — 来源为大单统计，与交易所精度有差异

## 时序注意（2026-05-26测试校正）

**10jqka页面两个区域有不同的时效性：**

| 区域 | 时效 | 适用时段 |
|------|------|---------|
| 顶部"实时资金流向" | 盘中不显示（"--"） | 仅盘中登录后可用 |
| "近30个交易日历史数据"表格 | **收盘后即更新当日数据** | 15:30后可用，含当日资金流向+涨跌幅+多维度拆分 |

2026-05-26 15:30+实测：300410页面"近30个交易日历史数据"表格已包含当日的完整数据行（20260526收盘11.43 +0.70%，主力净流入+669.52万，占比+1.06%）。

**铁律**：交易时段内(9:30-15:00)用此页面只能拿到历史数据。收盘后15:30+可拿到当日完整资金流向。

## MCP限流失效降级链（2026-05-26验证）

当iFinD MCP返回"用户使用工具已超限"（临时限流，非永久403 quota exhausted）时，推荐降级链：

```
Step 1: 等2分钟后重试MCP
  ❌ 仍限流 → Step 2

Step 2: 10jqka静态页面（首选降级）
  URL: http://stockpage.10jqka.com.cn/{code}/Funds
  返回: 近30日历史数据表格，含当日（收盘后）或前几日数据
  web_extract 直连可用，无VPN依赖
  字段: "大单(主力)净额"(万元)、"大单占比"(%)
  ✅ 可用 → 返回
  ❌ 不可用 → Step 3

Step 3: web_search 搜索"股票名 资金流向 日期"
  证券之星、网易财经等新闻文章有时有单日汇总
  但口径不同平台间可能不一致
```

## 对比其他降级方案

| 降级方案 | 时效性 | 可靠性 | 数据精度 | 适用维度 |
|----------|--------|--------|---------|----------|
| 10jqka资金流向页面 | T+0（盘后更新） | ✅ 静态HTML | 中（万为单位） | 主力净流入 |
| push2 f162/f164 | 实时 | ⚠️ 字段受限 | 高（手数+占比） | 主力净流入（补充） |
| 证券之星新闻文章 | T+0 | ⚠️ 不可控 | 低（口径可能不同） | 单日资金流向 |
| web_search + 交叉验证 | 最近一个月 | ⚠️ 不可控 | 低（需跨源对比） | 龙虎榜/大宗交易（只有/无） |
