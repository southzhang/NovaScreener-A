# 搜狐证券融资融券数据降级方案

## 场景

当 iFinD MCP 额度耗尽 + push2/东方财富 API 被 VPN 拦截时，搜狐证券 `q.stock.sohu.com` 作为融资融券数据降级源。

## 查询方式

### 单股融资融券明细

```
https://q.stock.sohu.com/app2/mpssTrade.up?code={code}&sd=&ed=
```

参数：
- `code` 是纯数字代码（如 300065）
- `sd` 和 `ed` 是日期范围（留空 = 近30天）

### 调用方式

web_extract 直接抓取，100% 可用：

```
web_extract(["https://q.stock.sohu.com/app2/mpssTrade.up?code=300065"])
```

## 返回数据

获取到一个完整的 HTML 表格，web_extract 会解析为 markdown 表格，包含：

| 字段 | 含义 | 单位 |
|------|------|------|
| 融资余额 | 当日收盘融资余额 | 元 |
| 融资买入 | 当日融资买入额 | 元 |
| 融资偿还 | 当日融资偿还额 | 元 |
| 融券余额 | 当日收盘融券余额 | 元 |
| 融券卖出(股) | 当日融券卖出股数 | 股 |
| 融券偿还(股) | 当日融券偿还股数 | 股 |
| 融券余量(股) | 当日融券余量 | 股 |
| 融券余量金额 | 融券余量 x 收盘价 | 元 |
| 融资融券余额 | 融资余额 + 融券余额 | 元 |

## 优点

1. **无需 VPN/代理** — web_extract 直连，不受 VPN 限制
2. **结构化数据** — 返回完整表格，web_extract 自动解析成 markdown
3. **历史完整** — 近30天每日数据，可看趋势
4. **高可靠性** — 2026-05-26 实测 300065（海兰信）返回完整18行日频数据

## 局限性

1. **仅支持融资融券** — 不提供主力净流入、龙虎榜等其他资金面数据
2. **单股查询** — 一次只能查一只，无法批量
3. **不含当日盘中数据** — 数据是收盘后更新，盘中查询到的是前一交易日数据
4. **融资偿还列可能显示为0** — 搜狐源的数据格式问题，融资偿还可能没有正确传递（但融资余额趋势仍可判断）

## 实测结果

### 海兰信(300065) — 2026-05-26

web_extract 返回了 2026-04-27 至 2026-05-25 共18行完整日频数据，包括：
- 融资余额从 10.86亿 到 11.45亿 的趋势
- 融券余额 ~400万（极低）
- 融资买入每天 1.1亿~3.1亿
- 数据清晰可读，可直接用于分析

## 与其他降级源的对比

| 数据源 | 数据质量 | 历史深度 | VPN兼容 | 时效性 |
|--------|---------|---------|---------|-------|
| 搜狐证券 | 高（结构化表格） | 近30天 | ✅ 兼容 | 收盘后更新 |
| 凤凰网财经 | 中（摘要文本） | 仅最新 | ✅ 兼容 | ⚠️ 可能为历史快照 |
| push2 东方财富 | 最高（JSON） | 近30天 | ❌ VPN拦截 | 盘中实时 |

## 推荐优先级

当 MCP 额度耗尽且需要融资融券数据时：

1. 优先 web_extract 搜狐证券（结构化、历史完整）
2. 备选 web_extract 凤凰网财经（快速定性判断）
3. 不使用 push2（VPN 下不可靠）
