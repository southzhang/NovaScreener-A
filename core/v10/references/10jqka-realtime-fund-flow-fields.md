# 同花顺实时API (d.10jqka.com.cn) — 资金流向字段映射（2026-05-26实测）

## 概述

同花顺实时API `d.10jqka.com.cn/v2/realhead/hs_{code}/last.js` 是一个JSONP接口，返回实时的详细行情数据，**包含资金流向分类字段**（超大单/大单/中单/小单的流入流出金额）。

**核心优势**：
- 在Shadowrocket VPN拦截下，`curl --noproxy "*"` 直连可用
- 无需认证、免费、不限量
- 包含腾讯行情API没有的资金流向分类数据
- 盘后可用，收盘后即更新当日完整数据

## 接口格式

```
https://d.10jqka.com.cn/v2/realhead/hs_{code}/last.js
```

- `{code}`: 股票代码，如 `300410`（深交所）或 `600936`（上交所）
- 返回JSONP：`quotebridge_v2_realhead_hs_{code}_last({...})`

## 调用方式

```bash
# 绕过VPN代理，直连成功（2026-05-26实测）
curl -s "https://d.10jqka.com.cn/v2/realhead/hs_300410/last.js" --connect-timeout 5 --max-time 8 --noproxy '*'
```

```

超大单净额 = 223 - 224
大单净额 = 225 - 226
中单净额 = 237 - 238
小单净额 = 259 - 260
```

## 与stockpage资金流向页面的关系

| API | 特点 | 优劣 |
|-----|------|------|
| `d.10jqka.com.cn` 实时API | JSONP，程序化获取，字段原始 | 可编程，字段无单位说明 |
| `stockpage.10jqka.com.cn/{code}/funds/` 静态页面 | HTML表格，万元为单位，已做分类 | 人类可读，但需web_extract解析 |

**推荐组合**：
- 程序化获取资金流向 → 实时API (d.10jqka.com.cn)
- 人类阅读/确认 → 静态页面 (stockpage.10jqka.com.cn)
- 两者数据口径应一致，如有差异以静态页面为准（单位万元，字段名明确）

## 局限性

- **JSONP格式**：需去掉函数包裹 (`data.split("=",1)[1].slice(0,-1)` 再 `JSON.parse`)
- **字段无文档**：字段含义通过交叉验证推断，非官方文档
- **无字段名**：只有数字key，需手动映射
- **单只查询**：不支持批量，每只股票单独请求
- **频率限制**：未测试，但建议不要超过10次/秒

## 适用场景

1. **逐只深度资金分析**：对少数持仓/候选股逐只获取完整资金流向分类
2. **iFinD MCP限流时降级**：当MCP返回"用户使用工具已超限"时，此API可替代
3. **push2被墙时替代**：当push2.eastmoney.com返回exit 52时，此API可用
4. **交叉验证**：与MCP/push2数据对比，确认资金流向方向

## 不适用场景

1. **批量查询(>3只)**：每只单独请求，效率低
2. **历史资金流向(>1日)**：此API只有当日数据，历史数据需用stockpage静态页面
3. **龙虎榜/大宗交易/融资融券**：此API不含上述数据
