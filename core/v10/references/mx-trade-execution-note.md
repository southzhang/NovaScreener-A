# 模拟盘交易执行注意事项（实践记录）

## 市价买入的执行细节
1. **非交易时间也可以提交委托**（09:28提交的市价单被系统接受），但委托会进入"已报待成"状态，直到9:30开盘后才撮合成交
2. **市价委托价**：提交时系统根据集合竞价价确定委托价（如光迅科技147.16元），如果开盘后价格快速跳空高开，原委托价过低会导致无法成交
3. **撤单再买可行**：如市价单未能及时成交，可先撤单再重新提交"市价买入"
4. **最终成交价**：模拟盘市价单的成交价是开盘后的实际撮合价，光迅科技从委托价147.16撤单后重新提交，最终以约150.60元左右成交

## 实时行情获取
- **妙想数据在开盘前09:27确实可以获取今日数据**（entityName显示当日时间戳），不是严格的"1-3天延迟"
- 但是"最低价=最高价=最新价"时表示集合竞价还未完全形成

## 盈亏确认
- 浮动盈亏以raw json中的`dayProfit`和`profit`字段为准
- 百分比要除以100（如-1.7393 = -1.74%）
