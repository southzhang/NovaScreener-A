## 数据源架构（当前正确状态）

### 全市场扫描 (v10_realtime_scan.py)
```
全市场实时行情 → 腾讯 qt.gtimg.cn（免费，~4290只）
条件预筛选 → iFinD 问财 smart_stock_picking
精确行情(PB) → iFinD real_time_quotation（500只/批）
日K线(250日) → 腾讯 web.ifzq.gtimg.cn（前复权）
分钟K线(今日) → iFinD high_frequency（信号股）
板块排名 → iFinD 问财板块分析
深度分析 → iFinD cmd_history_quotation
```

### 盯盘监控 (v10_intraday_monitor.py)
```
持仓行情 → iFinD real_time_quotation（首选）+ 腾讯 qt.gtimg.cn（备用）
盘口快照 → iFinD snap_shot（最近15分钟tick）
```

### 持仓监控 (intraday_holdings_monitor.py)
```
行情 → 腾讯 qt.gtimg.cn（HTTP，非HTTPS）
```

### 模拟盘交易 (v10_auto_trade.py)
```
股票列表 → 新浪 HTTP API
K线数据 → 新浪 HTTP API
交易执行 → mx-moni API（东方财富模拟盘）
```

### Cron Agent任务数据源
```
精确行情 → iFinD HTTP API real_time_quotation
基本面 → iFinD MCP工具（5项验证）
财务趋势 → iFinD HTTP API date_sequence
板块成分 → iFinD HTTP API data_pool (p03291)
```
