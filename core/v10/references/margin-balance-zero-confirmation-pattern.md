# 融资融券余额为零的确认模式

## 场景描述
当需要查询某只A股的融资融券余额，但所有数据源均返回空/零时，需要区分：
1. **数据源问题** — API不可用、域名被拦、字段名变化
2. **确实无余额** — 该股虽然被纳入融资融券标的，但从未产生融资融券交易余额

## 诊断流程

### Step 1: 确认该股是融资融券标的

**首选方法（推荐，无需VPN/无网络限制）：** 通过 iFinD MCP 直接查询
```python
mcp_ifind_stock_get_stock_info(query="300410正业科技是否属于融资融券标的")
# 返回: 是否融资融券标的 → 是/否
# 如果返回"否"→ 空值正确，终止诊断。后续所有融资余额查询都会为空，不是数据源问题。
# 如果返回"是"→ 继续 Step 2 跨源验证
```
2026-05-27实测：正业科技(300410)返回"否"→ 非标的 → 全部空值正确。**比push2 f107更快更可靠，无VPN依赖。**

**方法B（备选）:** 通过东方财富 push2 检查 f107 字段
```python
# 0=非标的, 1=标的
curl -s --noproxy "*" "https://push2.eastmoney.com/api/qt/stock/get?secid=0.300473&fields=f107&fltt=2&ut=fa5fd1943c7b386f172d6893dbbd4848"
# ⚠️ push2 API 在 VPN/VPS 环境下可能 RemoteDisconnected 不可用
```

### Step 2: 跨源验证余额
三到四个独立数据源，全部空才确认"无余额"：

| 数据源 | 查询方式 | 零标志 | 可靠性 |
|--------|----------|--------|--------|
| 搜狐证券 (q.stock.sohu.com) | web_extract 直接抓取表格 | 显示完整表格（近30天日频数据） | ✅ 稳定可抓，无VPN依赖 | 2026-05-26实测海兰信300065返回18行完整数据 |
| 同花顺 iFinD MCP | `get_stock_performance(query="300410正业科技融资余额")` 或 `get_stock_shareholders(query="300410正业科技的融资余额")` | 返回空/制表符 | ⚠️ iFinD MCP 对小盘股/零余额股经常返回空值（已知数据格式不稳定问题），不能单靠此判断 |
| 搜狐证券 (q.stock.sohu.com) | web_extract 抓取历史表格 | 显示完整近30天融资融券明细 | ✅ 稳定可靠，数据结构化强 | 融资偿还列可能显示为0（搜狐源格式问题），但不影响融资余额趋势判断 |
| 凤凰网财经 (finance.ifeng.com) | web_extract 页面摘要中的融资融券余额字段 | 显示0元或非零数值 | ✅ 页面稳定可抓  数据可能陈旧 -- 2026-05-26实测海兰信(300065)页面的资金流向数据是2024-02-02的，财务数据停留在2023年。融资余额显示6.16亿但很可能不是当日最新值。页面是历史快照并非实时刷新。不能用于确认当日最新融资余额，仅能作为确有/无余额的定性判断。 |
| 东方财富网页 | `web_extract("https://data.eastmoney.com/rzrq/stock/{code}.html")` | 显示"暂无相关数据" | ⚠️ 动态加载，web_extract可能抓不到具体数值，只能看到占位文字 |
| 东方财富 datacenter-web RZRQ | `reportName=RPTA_WEB_RZRQ_GGMX&columns=DATE,SCODE,SECNAME,RZYE,RZRQYE&filter=(SECUCODE%3D%220.300473%22)` | filter返回空列表 | ❌ 2026-05确认多数RZRQ报表已返回9501"报表配置不存在" |

**顺序优先级**: 凤凰网财经 → 东方财富网页 → iFinD MCP（因MCP常返空值，不优先依赖）|

### Step 3: 确认原因
余额为零的可能原因：
- **刚纳入（1-3个月内）** — 券商才公布上调，券商客户尚未开始融资买入
- **小市值/低换手** — 融资融券标的中成交不活跃的，券商可能未提供券源或杠杆率低于普通标的
- **股东质押比例高** — 可出借的券源不足
- **股价波动大/高风险** — 券商风控限制融资比例

### 数据源铁律
1. datacenter-web 的 RZRQ 报表 **返回空不是API故障**，是该股在数据库中无任何融资融券余额历史记录
2. 融资融券余额为零 ≠ 数据源问题。需要三个独立源交叉验证后确认无余额
3. 融资融券余额为零 ≠ 该股不是融资融券标的。两件事是独立的
4. 凤凰网数据时效性陷阱：凤凰网页面返回的融资余额数值可能是历史快照（页面上其他数据如资金流向、财务数据停留在数月前）。融资余额值虽然显示非零，但不一定是当日最新值。定性判断（有余额/无余额）可靠，定量数值需谨慎。

## 凤凰网数据时效性诊断方法

当凤凰网返回非零融资余额时，需判断数据是否最新：
1. 检查页面中其他数据的日期：资金流向数据日期、财务数据报告期。如果资金流向停留在2024年、财务数据显示2023年，则融资余额也可能不是最新的。
2. 交叉验证行情数据：凤凰网页面显示的昨收价/涨跌幅是否与今日行情一致。不一致则页面整体为历史快照。
3. 结论分级：
   - 页面有当日行情数据（涨跌幅吻合） -> 融资余额可信为最新
   - 页面显示历史行情/财务数据 -> 融资余额仅为参考值，需其他源确认
   - 页面显示0元 -> 零余额定性可靠，无论是否为快照

## 从empty值到结论的诊断API响应速查（2026-05-26补充）

当融资余额查询返回空值时，各API的具体错误/空响应模式可作为快速诊断依据：

| 数据源 | 查询方式 | 空/错误响应 | 含义 |
|--------|----------|------------|------|
| iFinD MCP get_stock_shareholders | `query="300410正业科技的融资余额"` | 表字段为空制表符，`code:1 success` | 非标的/无数据，接口正常 |
| iFinD MCP get_stock_performance | `query="300410正业科技最新融资余额"` | 同上，表字段为空制表符 | 同上 |
| East Money push2 RZRQ | `secid=0.300410&fields=f43...&fltt=2` | `{"rc":102,"data":null}` | rc:102=无可用数据，非API故障 |
| Sina finance margin API | `getRZRQData?symbol=300410` | `{"__ERROR":3,"__ERRORMSG":"Service not found"}` | 该股不在Sina融资融券数据库 |
| Sina finance hq.sinajs.cn | `list=sz300410` | 返回GBK行情，无融资字段 | 基础行情API不含融资融券维度 |
| mx-data (东方财富妙想) | `python3 mx_data.py "300410融资余额"` | `错误: 接口返回中无 dataTableDTOList` | mx-data不支持融资融券指标查询 |
| 腾讯行情/腾讯K线 | qt.gtimg.cn / ifzq.gtimg.cn | 正常返回基础行情和K线，无融资融券字段 | 腾讯API不提供融资融券维度 |
| 凤凰网财经 | web_extract | 页面静态显示"融资融券余额: 0元" | 最可靠零余额确认源 |

**铁律**：看到 `rc:102` 或 `"Service not found"` 说明该股确实无融资融券数据，不用再试其他付费/免费数据源。凤凰网页面是最快的"终结者"——一次web_extract确认即可下结论。

## 参考案例

### 案例1：正业科技(300410) — 2026-05-27验证
- 股票属性：创业板小盘股(~43亿市值)，PCB/锂电检测设备
- **所有查询路径结果**：
  - iFinD MCP 融资余额字段 → 空值
  - 凤凰网财经 → "融资融券余额: 0元（融资0元，融券0元）"
  - 东方财富 → "暂无相关数据"
  - East Money push2/datacenter → 全部被拦
- **最终结论**：该股融资融券余额确实为0——可能非标的，或虽是标的但从未产生融资融券交易
- **关键教训**：当 iFinD MCP 返回空值时，凤凰网财经 web_extract 是最快确认的数据源

### 案例2：德尔股份(300473) — 2026-05-22验证
- 2026年3月长城证券将其纳入融资融券标的名单（来源：东方财富公告搜索）
- 截至2026年5月22日，所有数据源余额均为零
- 推断：小市值(25亿)、低换手(2%左右)、刚纳入2个月，尚未产生融资融券交易

### 案例3：海兰信(300065) — 2026-05-26验证
- 股票属性：创业板中盘股(~130亿市值)，海洋电子/军工概念
- 标的资格：是融资融券标的（iFinD MCP确认）
- **查询路径结果**：
  - iFinD MCP get_stock_shareholders -> 表字段为空（制表符）
  - iFinD MCP get_stock_performance -> 表字段为空（制表符）
  - push2 API -> VPN下全部RemoteDisconnected
  - 凤凰网财经 -> 显示融资余额61,632.64万元，融券余额293.94万元
- **关键发现（时效性陷阱）**：凤凰网页面显示的资金流向数据日期为2024-02-02，财务数据为2023年。页面整体是一个历史快照，融资余额6.16亿很可能不是当日最新值。
- **教训**：凤凰网数据可靠用于定性判断（有余额/无余额），定量数值可能陈旧。页面中其他数据（资金流向日期、财务报告期）的时效性是判断融资余额新鲜度的关键线索。
