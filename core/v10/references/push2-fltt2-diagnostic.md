# push2 API fltt=2 诊断备忘

## 症状速查

当调用 `push2.eastmoney.com/api/qt/stock/get` 时：

| 症状 | 病因 | 修复 |
|------|------|------|
| f62=1 | 漏了 fltt=2 | 加 `&fltt=2` |
| f84=f85 完全相等（如都是367114797） | 同上 | 同上 |
| f184=37.85（很小的小数） | 同上 | 同上 |
| f190=-3.42 | 这个可能是真实的占比值（即使没有fltt=2也可能返回） | — |
| curl返回空响应 exit 52 | 被VPN拦截，或SSL握手失败 | 加 `--noproxy "*"` 绕过代理 |
| 价格字段f43=1140（不是11.40） | 未除100（东财价格以分为单位） | `price = f43 / 100` |

## 正确调用姿势

```bash
# ✅ 正确的完整调用
curl -s --noproxy "*" \
  "https://push2.eastmoney.com/api/qt/stock/get?secid=0.300410&fields=f62,f84,f85&fltt=2" \
  --connect-timeout 5
```

## fltt=2 字段映射（实测）

| 字段 | 含义 | fltt=2 时 | 无 fltt=2 时 |
|------|------|-----------|-------------|
| f62 | 主力净流入(元) | 实际金额，可正可负 | 返回1（flag：有数据） |
| f84 | 主力买入额(元) | 实际买入金额 | 返回总成交额相关值 |
| f85 | 主力卖出额(元) | 实际卖出金额 | 返回值同f84 |
| f184 | 超大单净流入(万元) | 实际金额 | 返回值异常小 |
| f190 | 主力净占比(%) | 实际百分比 | 可能仍返回实际值（部分字段不受影响） |
