# iFinD 问财智能选股 API (smart_stock_picking)

## 概述
同花顺问财(iWenCai)智能选股的HTTP API版本。支持自然语言条件查询，返回匹配的股票列表+指标数据。

## 端点
```
POST https://quantapi.51ifind.com/api/v1/smart_stock_picking
```

## 认证
使用iFinD HTTP API的access_token（与MCP的JWT不同）。

## 参数
| 参数 | 类型 | 说明 |
|------|------|------|
| searchstring | string | 自然语言选股条件 |
| searchtype | string | "stock" 或 "fund" |

## 自然语言条件示例

### 基础条件
- `"涨幅前10"` — 涨幅排名前10
- `"ROE大于15%"` — ROE筛选
- `"市盈率小于30"` — PE筛选
- `"成交额大于5000万"` — 成交额筛选
- `"股价5元到150元"` — 价格区间

### 组合条件（V10预过滤推荐）
- `"涨幅0.5%到9.8% 成交额大于5000万 股价5元到150元 非ST 非北交所"`
- `"ROE大于10% 净利润增速大于0 市盈率小于50"`
- `"MACD金叉 放量"`

### 排除条件
- `"非ST"` — 排除ST股
- `"非北交所"` — 排除北交所

## 返回格式
```json
{
  "errorcode": 0,
  "tables": [{
    "table": {
      "股票代码": ["600756.SH", ...],
      "股票简称": ["保变电气", ...],
      "涨跌幅:前复权[20260513]": ["5.23", ...]
    }
  }]
}
```

## 耗时与Quota
- 简单查询：0.3秒，组合条件：0.6-0.9秒
- 月quota远大于实时行情API，适合频繁调用

## 注意事项
- 返回的股票代码格式：`600756.SH`，需转纯数字
- 条件太苛刻时可能返回空
- 需处理`errorcode != 0`的异常情况
