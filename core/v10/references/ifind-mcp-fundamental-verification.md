# iFinD MCP工具选股基本面验证规范

## 必须执行的5项查询（每个候选股）

### 1. 财务数据查询
```
工具: mcp_ifind_stock_get_stock_financials
查询: {"query": "XX股票 最新季度 ROE 净利润增速 营收增速"}
判断: ROE>8%且净利润增速>0%→✅ | 否则→❌
```

### 2. 事件信息查询
```
工具: mcp_ifind_stock_get_stock_events
查询: {"query": "XX股票 近30天 事件信息"}
判断: 有减持/诉讼→❌一票否决 | 有利好→⭐加分
```

### 3. 新闻舆情查询
```
工具: mcp_ifind_news_search_news
查询: {"query": "XX股票 最新新闻", "size": 5}
判断: 正面新闻→⭐ | 负面新闻→❌ | 无新闻→中性
```

### 4. 公告查询
```
工具: mcp_ifind_news_search_notice
查询: {"query": "XX股票 近期公告", "size": 5}
判断: 减持公告→❌ | 业绩预增→⭐ | 重大合同→⭐
```

### 5. 股东信息查询（14:30任务）
```
工具: mcp_ifind_stock_get_stock_shareholders
查询: {"query": "XX股票 前十大股东 持股变化"}
判断: 机构增持→✅ | 大股东减持→❌
```

### 6. 股票基本信息
```
工具: mcp_ifind_stock_get_stock_info
查询: {"query": "XX股票 基本信息 行业分类 主营业务"}
```

### 7. 热点事件查询（14:30任务）
```
工具: mcp_ifind_news_search_trending_news
查询: {"keyword": "板块名", "size": 5, "time_scope": "24小时"}
判断: 板块有持续性→⭐ | 一日游行情→⚠️
```

## 综合判断标准

| 合格项数 | 判定 |
|----------|------|
| 5项全合格 | ✅ 强推 |
| 4项合格 | ✅ 通过 |
| 3项以下 | ❌ 不推荐 |
| 有减持/诉讼 | ❌ 一票否决 |

## 各任务查询差异

| 任务 | 查询项数 | 特殊查询 |
|------|----------|----------|
| 09:25竞价 | 4项 | 基本信息 |
| 11:00盘中 | 4项 | 基本信息 |
| 14:30尾盘 | 5项 | +股东信息+热点事件 |
