# 量化盯盘选股软件 - Quant Watchdog

## 项目概述
A股量化盯盘选股工具，基于 akshare 免费数据源，提供选股策略、实时行情监控、飞书预警通知。

## 技术栈
- **语言**: Python 3.10+
- **数据源**: akshare (免费A股数据)
- **Web框架**: Streamlit (快速出原型)
- **图表**: plotly (K线图、技术指标图)
- **通知**: 飞书 webhook (通过 requests 调用)
- **数据库**: SQLite (轻量，存自选股和策略配置)
- **任务调度**: APScheduler (定时跑策略)

## 核心功能

### 1. 选股策略引擎
内置策略（可扩展）：
- **均线突破**: 5日均线上穿20日均线
- **MACD金叉**: DIF上穿DEA
- **放量突破**: 成交量突破近20日均量的2倍
- **RSI超卖反弹**: RSI从30以下回升
- **布林带突破**: 价格突破布林带上轨
- **自定义策略**: 用户可写简单Python表达式

策略参数可配置，支持多策略组合（AND/OR逻辑）。

### 2. 实时行情监控
- 全市场扫描（沪深主板+创业板+科创板）
- 定时刷新（默认每5分钟，可配置）
- 盘中实时涨跌幅排行
- 板块热度监控

### 3. 自选股管理
- 添加/删除自选股
- 自选股分组（如：短线、中线、观察）
- 自选股实时行情面板

### 4. K线图表
- 日K/周K/月K线
- 叠加均线（MA5/10/20/60）
- 成交量柱状图
- MACD/KDJ/RSI 副图

### 5. 飞书预警通知
- 选股策略触发时推送飞书消息
- 自选股异动提醒（涨跌超5%、涨停跌停）
- 支持配置多个webhook
- 消息格式：卡片消息，包含股票代码、名称、触发策略、当前价格

### 6. Web界面 (Streamlit)
页面结构：
- **首页/Dashboard**: 市场概览、今日策略信号、自选股行情
- **选股扫描**: 选择策略 → 运行扫描 → 查看结果
- **自选股**: 管理自选股列表、查看K线
- **策略配置**: 编辑策略参数、创建自定义策略
- **预警设置**: 配置飞书webhook、设置提醒规则
- **回测**: 简单回测功能（策略历史表现）

## 数据存储
SQLite表结构：
- `watchlist`: 自选股列表 (code, name, group, added_at)
- `strategies`: 策略配置 (name, params_json, enabled)
- `signals`: 策略触发记录 (code, strategy, triggered_at, price)
- `alerts`: 预警记录 (code, type, sent_at)

## 配置
- `.env` 文件存放敏感配置（飞书webhook URL等）
- `config.yaml` 存放策略参数、刷新间隔等

## 启动方式
```bash
# 安装依赖
pip install -r requirements.txt

# 启动
streamlit run app.py
```

## 注意事项
- akshare 接口有频率限制，需要做请求缓存
- 盘中数据延迟约3-5秒（免费数据源的限制）
- 回测需要先下载历史数据
- 飞书webhook需要用户自己创建并配置
