# 📊 NovaScreener 量化盯盘选股 （A股）

**维加斯强庄策略 · 七条件共振 · 波段回调 · 多维评分 · 盯盘监控**

> A股量化盯盘选股工具，基于腾讯免费数据源，内置强庄策略引擎 + 趋势波段策略 + 竞价选股 + 持仓顾问。

---

## 🚀 快速启动

```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 复制配置
cp .env.example .env
# 编辑 .env 填入飞书 webhook URL（可选）

# 3. 启动
streamlit run app.py
```

**访问 `http://localhost:8501` 即可使用。**

## ✨ 核心功能

### 🎯 七条件共振选股
核心策略引擎，从7个维度筛选强庄股：
- 量价配合、均线排列、MACD状态、KDJ超卖反弹
- 趋势强度、回调深度、资金动量
- 支持多策略并行扫描（全市场 5000+ 股票）

### 📈 趋势波段策略
趋势确认 + 回调入场 + 三层波段止盈：
- ✅ 多头排列（EMA20 > EMA50 > EMA120）
- ✅ 回调到支撑 + 缩量 + 动量未死
- ✅ ATR 动态止盈止损
- ✅ 三层分批出场策略

### ⚡ 竞价选股（09:25）
09:25 集合竞价结束后秒出结果：
- 趋势共振 / 游资爆量 / 游资竞价 三种子策略
- 板块热度过滤
- V10 交叉标记

### 💼 持仓顾问
基于趋势状态 + V10信号 + ATR 的动态操作建议：
- 🟢 持有不动 / 🟡 减仓 / 🔴 清仓
- 🔵 加仓 / 📈 动态止盈 / 💰 盈利离场
- 紧急程度 1-5 级

### 📊 综合评分系统
V10 六维评分 + 趋势评分 + 波段评分：
- 得分 0-100，自动排序
- 多维雷达图可视化

### 🔔 飞书预警通知
- V10信号触发时推送飞书卡片消息
- 支持自定义 webhook

### 🕒 盯盘监控
- APScheduler 定时全市场扫描
- 自选股实时行情刷新
- 盘中异常提醒

## ⚙️ 配置说明

### `config.yaml`
- `v10_params` — V10 评分各维度权重
- `scan_interval` — 全市场扫描刷新间隔（分钟）
- `monitor_interval` — 盯盘监控刷新间隔（秒）
- `top_n` — 扫描排名前N只

### `.env`
```
FEISHU_WEBHOOK_URL=https://open.feishu.cn/open-apis/bot/v2/hook/xxx
```
